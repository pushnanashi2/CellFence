# @cellfence/engine

Repository analysis engine for CellFence. It validates cell ownership, public surfaces, declared dependencies, artifact lanes, resource access, baselines, and runtime resource evidence. It also projects per-cell agent context so tools can read the allowed fence before editing.

Most users should install the `cellfence` CLI package instead.

See the main CellFence README: https://github.com/pushnanashi2/CellFence#readme
